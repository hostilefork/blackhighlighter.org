This page is designed to demonstrate that blackhighlighter really is a *widget*. While the blackhighlighter demo sandbox belabors the points of how the different phases work by putting them on six separate tabs and pages, you can use Ajax and never leave a single page. And you can have multiple blackhighlighter widgets on a page, as you might any textual div. Hence, the generality of the tool.

This is a one-page demo that lets you make as many blackhighlighter widgets as you like, commit them... verify certificates, and reveal them. It does all that without needing any more than a single URL. It shows the basis for a discussion system.

Note: This website is a demonstration sandbox for using the blackhighlighter jQuery plugin. The database is regularly erased—which means your post might not be here tomorrow! If you want a more persistent service, then install the software yourself, or get in touch with brian@hostilefork.com about joining forces to build blackhighlighter.com
